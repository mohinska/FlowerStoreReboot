# Flower Store REST API

A simple Spring Boot REST API for managing a flower store.

## Live Demo

* **Hosted on Heroku:** [https://flower-store-reboot-4abaf6016485.herokuapp.com/api/flowers](https://flower-store-reboot-4abaf6016485.herokuapp.com/api/flowers)

##  API Endpoints

**Base URL:** `https://flower-store-reboot-4abaf6016485.herokuapp.com`

### Flowers
* `GET /api/flowers`
* `POST /api/flowers`

### Users
* `GET /api/users`
* `POST /api/users`
